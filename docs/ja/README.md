## 現在の主な機能

- スレッド一覧と詳細ページ
- 親投稿の画像投稿
- 返信ではSNS・画像投稿を扱わない
- 固定文返信「ええじゃないか」
- トップページにスレッドごとの返信を最大10件プレビュー表示
- 親投稿だけに表示番号を付け、返信はスレッド内の子番号を使う
- 編集とソフトデリート
- 管理画面からの復元とパスワード不要の一括削除
- 管理画面のDB整合性チェック
- 管理画面のバックアップエクスポート/インポート
- Web管理画面ではなくローカル運用バッチから行うローカルアーカイブログの非破壊インポート
- 全文、タイトル、本文、作者での検索
- RSS出力
- X、Bluesky、Mastodon、Misskey の投稿設定。デフォルトではOFF
- SNSごとの転記プレビューと、文字数超過時の `..` 省略
- SNS転記は新規の親投稿作成時のみ。返信と編集時はSNSへ投稿しない
- X、Bluesky、Mastodon、Misskey への画像転記
- SNS投稿には `/#post-000001` のような掲示板一覧への「最新はこちら」リンクを含める
- SNSリアクション数は管理画面、ローカルCron、APIキー付き外部定期実行URLから更新できる
- gdgd 表示モード
- 組み込みマニュアルページ
- 管理画面から HOME リンク、マニュアル本文、SNSスイッチ、gdgdスイッチ、gdgdラベル、基本色を設定できる
- SNS認証情報欄は、各SNSがOFFの間は無効化される
- 管理者パスワード変更
